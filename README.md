# Acabou a Pipoca - Links

Pagina estatica de links recentes usada na bio do Instagram `@acabouapipoca`.

## Estrutura publica

- `docs/index.html`: pagina publicada pelo GitHub Pages.
- `docs/links.json`: links recentes exibidos na pagina.
- `docs/logo.jpg`: logo usado no topo.
- `docs/.nojekyll`: desativa processamento Jekyll.

A automacao do projeto privado atualiza somente estes artefatos publicos.
